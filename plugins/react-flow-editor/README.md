# react-flow-editor

Expert skill for building interactive node-based diagram editors with React Flow (@xyflow/react). Flow diagrams, network topologies, data pipelines.

## Category

UI / Visualization

## Installation

```
/plugin marketplace add Maxymize/maxym-claude-plugins
/plugin install react-flow-editor@maxym-skills
```

## Usage

Once installed, this skill activates automatically when Claude Code detects a relevant task. See the `skills/react-flow-editor/SKILL.md` for complete documentation and examples.
