# skill-creator

Guide for creating effective Claude Code skills that extend capabilities with specialized knowledge, workflows, or tool integrations.

## Category

Meta / Builder

## Installation

```
/plugin marketplace add Maxymize/maxym-claude-plugins
/plugin install skill-creator@maxym-skills
```

## Usage

Once installed, this skill activates automatically when Claude Code detects a relevant task. See the `skills/skill-creator/SKILL.md` for complete documentation and examples.
