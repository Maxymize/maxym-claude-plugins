# infrastructure-analyzer

Analyze project codebase and generate comprehensive infrastructure documentation in Markdown and HTML formats.

## Category

Documentation

## Installation

```
/plugin marketplace add Maxymize/maxym-claude-plugins
/plugin install infrastructure-analyzer@maxym-skills
```

## Usage

Once installed, this skill activates automatically when Claude Code detects a relevant task. See the `skills/infrastructure-analyzer/SKILL.md` for complete documentation and examples.
