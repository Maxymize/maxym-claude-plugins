# mcp-builder

Guide for creating high-quality MCP (Model Context Protocol) servers that enable LLMs to interact with external services. Supports Python (FastMCP) and Node/TypeScript.

## Category

Meta / Builder

## Installation

```
/plugin marketplace add Maxymize/maxym-claude-plugins
/plugin install mcp-builder@maxym-skills
```

## Usage

Once installed, this skill activates automatically when Claude Code detects a relevant task. See the `skills/mcp-builder/SKILL.md` for complete documentation and examples.
