# coolify-code-exec

Direct Coolify REST API access for application management, deployments, servers, databases, and services. 99%+ token reduction.

## Category

DevOps & Hosting

## Installation

```
/plugin marketplace add Maxymize/maxym-claude-plugins
/plugin install coolify-code-exec@maxym-skills
```

## Usage

Once installed, this skill activates automatically when Claude Code detects a relevant task. See the `skills/coolify-code-exec/SKILL.md` for complete documentation and examples.
