# c15t-consent

Implement GDPR/CCPA-compliant consent management with c15t. Covers cookie banners, consent dialogs, and privacy compliance for React, Next.js, and vanilla JavaScript.

## Category

Privacy & Compliance

## Installation

```
/plugin marketplace add Maxymize/maxym-claude-plugins
/plugin install c15t-consent@maxym-skills
```

## Usage

Once installed, this skill activates automatically when Claude Code detects a relevant task. See the `skills/c15t-consent/SKILL.md` for complete documentation and examples.
