# code-execution-creator

Create Code Execution skills from existing MCP servers for 99%+ token reduction. Supports Direct Connection (HTTP API, database) and MCP Bridge approaches.

## Category

Meta / Builder

## Installation

```
/plugin marketplace add Maxymize/maxym-claude-plugins
/plugin install code-execution-creator@maxym-skills
```

## Usage

Once installed, this skill activates automatically when Claude Code detects a relevant task. See the `skills/code-execution-creator/SKILL.md` for complete documentation and examples.
